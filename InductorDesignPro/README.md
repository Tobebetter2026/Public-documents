# 电感设计专家

**电感设计专家** 是一款专业的电感设计工具，帮助电子工程师快速迭代计算和导出电感设计方案。

## 主要功能

- **多步设计流程**：从参数输入到最终报告，全程引导
- **智能计算**：自动优化线圈参数、磁芯选择
- **专业版导出**：支持 PDF、Excel格式的设计报告
- **iCloud 同步**：跨设备无缝使用
- **深色模式**：完美适配 iOS 视觉风格

## 使用说明

1. **第一步**：输入电感基本参数（电感值、电流等）
2. **第二步**：输入约束条件
3. **第三步**：迭代计算并优化磁芯线圈设计
4. **第四步**：查看汇总报告，导出设计文件

*部分高级功能需订阅专业版解锁。*

## 常见问题

### **1. 如何导出PDF/EXCEL设计报告？**
需使用**专业版**。在第四步「汇总与报告」页的「设计文件」卡片中，点击 **「导出PDF」或「导出EXCEL」** 按钮，浏览器会打开打印/另存为 PDF 或者 EXCEL 的对话框，选择保存位置即可。iOS 上可通过系统分享菜单保存到文件或发送。  
*免费版不支持导出，升级专业版后可解锁 PDF、JSON、Excel 导出。*

### **2. 专业版如何订阅？**
打开 App，进入「设置」→「升级到专业版」，选择年度或永久订阅方案。免费版每天可进行 3 次设计计算且功能受限（如不可导出、磁芯与材料种类有限）；购买专业版后立即解锁全部功能（无限次使用、全部磁芯与材料、多方案对比及 PDF/Excel 导出等）。

### **3. 是否支持离线使用？**
是的，核心计算功能完全离线运行。iCloud 同步需要网络连接。
