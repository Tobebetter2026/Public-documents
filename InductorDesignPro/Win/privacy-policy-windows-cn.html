隐私政策（Windows 版）

适用于通过 Microsoft Store 发行的「电感设计专家」（Inductor Design Pro）Windows 版

本隐私政策（下称「本政策」）适用于通过 Microsoft Store 分发的「电感设计专家」Windows 版应用程序（下称「本应用」、「我们」）。本政策说明我们会如何处理与您的设备相关的信息。请在使用前阅读。

1. 引言

「电感设计专家」为 Buck、Boost、Forward、交错 Buck（多相） 等功率电感的本地计算与设计辅助工具；界面提供中文与英文显示，语言偏好主要依据您设备的系统语言设置，不改变本政策所述的数据处理原则。我们遵循隐私优先、计算与数据主要在设备本地完成的原则：您的设计内容与自建数据库默认仅存于您的设备；除非下文另有说明，我们不会收集可用于识别您身份的信息用于广告投放或与第三方共享。

2. 我们如何收集与使用信息

2.1 基本原则
本应用不向开发者服务器上传您的设计方案、规格参数或「我的数据库」内容；我们无法远程读取您设备内的上述数据。

2.2 在您设备上处理的信息
您输入或生成的数据包括但不限于：

  • 电气规格与设计约束（电压、电流、频率、拓扑与模式等）
  • 选型与迭代计算结果、汇总报告所含字段
  • 本地保存的项目与偏好设置
  • 「我的数据库」中的自定义材料、磁芯、骨架

您在设备上生成的 PDF 报告与 JSON 设计文件由本机在本地生成；若您使用系统分享、邮件或其他应用打开或发送，则由您选择的接收方按其条款处理。

上述信息仅用于在您的设备上呈现计算结果与导出文件，不会上传至开发者服务器。

2.3 分析与诊断（匿名）
我们可能通过 Microsoft 提供的标准机制（例如 Microsoft Store 分析、匿名崩溃诊断）获知设备型号、系统版本及聚合使用概况，不包含您的设计明细。您可在 Windows「设置 → 隐私和安全性 → 诊断和反馈」中管理诊断数据共享以限制此类上报。

3. 数据存储

设计数据主要保存在本机应用的本地数据目录。删除应用或清除数据将依系统行为删除本地内容。本应用不使用 iCloud 或其他云备份服务存储您的设计数据；若 Windows 将本应用的本地数据纳入您自行配置的系统级备份（例如文件历史记录），则由相应 Microsoft 服务的条款约束，本开发方无法访问此类备份。

本应用不保存本地试用开始时间作为唯一信任依据，也不仅凭本地时间戳解锁付费功能。7 天全功能试用由应用首次启动时在本机起算，与最终选择「年度订阅」还是「一次性买断」无关；年度订阅与一次性买断的购买记录、收据与（年度订阅的）自动续订均由 Microsoft（Microsoft Store / Partner Center）处理。应用在需要时通过官方 Windows.Services.Store 接口读取有效的订阅或购买状态，并可能在本机缓存该状态与到期时间（一次性买断为永久，无到期时间），用于界面展示和离线时的基础状态延续。

4. 数据共享

我们不出售设计数据。因应用发布、支付与系统服务需要，Microsoft（Microsoft Store、Partner Center、订阅与购买记录）及您主动选择的网络通讯对方（例如在浏览器中打开的隐私政策网页托管方）按其各自条款处理与该连接相关的最少必要信息。

5. 第三方与本应用网页

若您在浏览器中打开随应用提供的说明页或隐私政策页面，会通过 HTTPS 加载静态内容；不包含用于追踪您个人的嵌入式广告脚本。GitHub Issues 等外链由您在浏览器中自主选择是否访问。

6. 儿童隐私

本应用并非专为 13 周岁以下儿童设计；我们不会明知而收集儿童的个人信息。

7. 您的控制权

您可在应用内管理设计方案与数据库；卸载应用可删除本机数据。年度订阅的自动续订设置由您在 Windows「设置 → 应用 → 订阅」或 Microsoft Store「订阅」页面中管理；一次性买断为永久解锁，无需管理续订。购买记录与收据由 Microsoft（Microsoft Store）按其条款处理。

8. 政策变更

我们可能更新本政策；重大变更将通过 Microsoft Store 更新说明或应用内适当方式提示。继续使用即视为接受更新后的政策。

10. Windows 平台说明

本应用通过 Microsoft Store 以打包桌面应用（MSIX）形式分发，界面基于 WebView2 本地加载。无论您使用何种 Windows 设备，数据处理原则相同：设计与计算在您当前使用的设备上进行，本产品不向开发者传输您的设计文件内容。

应用无需位置、摄像头、麦克风等非必要权限即可使用核心功能；核心迭代与损耗计算可离线完成，打开 Microsoft Store 订阅/购买页面、校验购买状态或浏览网页帮助需要网络连接。

首次启动时，Windows 可能执行标准的应用完整性检查（例如 Microsoft Defender SmartScreen），这是平台官方安全机制的一部分，可能需要网络连接；此后应用可离线运行。设计项目存储在应用本地数据目录中，不与其他应用共享。

7 天全功能试用、年度订阅与一次性买断均通过 Microsoft 完成（包括 Microsoft Store 的订阅与购买规则，以应用内及 Microsoft Store 展示为准），我们不将订阅/购买信息用于第三方广告画像。
