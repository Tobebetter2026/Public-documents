电感设计专家

最后更新：2026-04-27

电感设计专家 为 Buck / Boost 功率电感提供从规格与约束、磁芯与绕组联合迭代，到汇总报告与导出的完整工作流；计算在设备本地完成，设计数据不向开发者服务器上传。

应用采用 14 天全功能试用，之后需通过 App Store 年度订阅继续使用完整能力；具体权限与试用期规则以应用内为准。

2.0版要点（相对早期版本）

  • 规格与约束合并为一步，集中预检
  • 磁芯 × 线规（含利兹）联合迭代、多方案对比与更完整的损耗/温升链路
  • 内置材料 / 磁芯 / 骨架库持续扩充；「我的数据库」新增批量导入/导出（按模板），自定义材料、磁芯、骨架在试用或订阅期内可用，并与内置库合并使用
  • 汇总页 PDF 报告与导出能力优化

主要功能概览

  • Buck / Boost 拓扑，CCM / DCM / BCM，AP 与 LI² 等选型辅助
  • 多种损耗模型路径（Steinmetz、厂商曲线/表、铁粉官方公式等，依材料类型选用）
  • 帕累托前沿筛选、多组备选方案对比；可选 iCloud 在您的设备间同步，设计主数据仍以本机为主
  • PDF / JSON 报告与设计文件导出
  • 「我的数据库」（本应用突出优势）：本机维护材料、磁芯、骨架，支持批量导入/导出，与内置库一并参与选型；不上传服务器。

使用流程（三步）

  1. 规格与约束：输入电气规格与设计约束，通过预检后进入下一步
  2. 磁芯与绕组：联合迭代选型、方案比较与确认
  3. 汇总与报告：损耗与温升等汇总、设计文件与报告导出

常见问题

  • 如何导出 PDF 或保存 JSON？在 14 天试用期内或有效年度订阅期间，于第三步「汇总与报告」操作；试用期满未订阅须先订阅（应用内提示）。
  • 如何开通订阅？打开应用内「设置」页，点击「订阅」；或在试用结束、导出 / 保存 JSON受限、进入第二步 / 第三步被提示等场景，点击「开启年度订阅」或「查看年度订阅」。均为同一订阅流程，通过 Apple 完成年度订阅。
  • 能否离线使用？核心计算可离线；iCloud 与 App Store 相关操作需网络。
  • 技术支持：GitHub Issues

1. 如何导出 PDF/Excel 设计报告？
需使用专业版。在第四步「汇总与报告」页的「设计文件」卡片中点击导出按钮。免费版不支持导出，升级专业版后可解锁 PDF、JSON、Excel 导出。

2. 专业版如何订阅？
打开 App，进入「设置」→「升级到专业版」选择订阅方案。

3. 是否支持离线使用？
是的，核心计算功能完全离线运行。iCloud 同步需要网络连接。
